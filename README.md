# AI-study-app
AI Study Assistant is a full-stack web application that allows users to securely create accounts, manage personal study notes, and get quick summaries using an AI-like feature.

The application includes user authentication with JWT, a backend built using Flask, and a frontend built with HTML, CSS, and JavaScript. Users can add, view, delete, and summarize notes, with all data securely tied to their account.

This project demonstrates full-stack development, REST API integration, authentication, and basic AI feature implementation.
